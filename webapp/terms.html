1. Aceptación

Al acceder, registrarte o utilizar KataraLM (la “Plataforma”), aceptas estos Términos y Condiciones (los “Términos”). Si no estás de acuerdo, no utilices la Plataforma.

Estos Términos regulan la relación entre tú (“Usuario”) y Wicca Inc. (“Operador”) respecto al acceso y uso de KataraLM, incluyendo la versión web y aplicaciones móviles.

2. Descripción del servicio

KataraLM es un asistente que ayuda a identificar residuos, aprender prácticas de reciclaje y encontrar puntos de acopio en Guayaquil. La Plataforma puede permitir:

  • Conversación con “Katara” para orientación de reciclaje y sostenibilidad.
  • Análisis de imágenes para identificar materiales y sugerir disposición.
  • Consulta de puntos de acopio y rutas/enlaces de navegación.
  • Gestión de perfil: biografía, foto, username y credenciales.
Objetivo: educación y orientación práctica. KataraLM no sustituye regulaciones oficiales ni asesoría técnica especializada.

3. Cuenta, registro y verificación

Para utilizar funciones completas, podrás crear una cuenta proporcionando correo, username y contraseña. Por seguridad, el registro incluye verificación mediante un código OTP enviado al correo.

3.1 Requisitos

  • Proporcionar información veraz y actualizada.
  • Elegir una contraseña segura y no compartirla.
  • No crear cuentas para suplantar a terceros.

3.2 Verificación por OTP

El OTP se utiliza principalmente para verificar tu correo en el registro y para recuperación de contraseña. El Operador podrá establecer límites de intentos y expiración del código.

4. Uso permitido y restricciones

Te comprometes a usar KataraLM de forma lícita y responsable.

4.1 Está prohibido

  • Usar la Plataforma para actividades ilegales o dañinas.
  • Intentar acceder sin autorización a cuentas, sistemas o datos.
  • Interferir con el funcionamiento del servicio (ataques, scraping abusivo, etc.).
  • Subir contenido que infrinja derechos de terceros o sea claramente ofensivo/ilegal.

4.2 Uso razonable

Para mantener la estabilidad, el Operador puede aplicar límites de tasa (rate limits), tamaños máximos de archivos, o restricciones temporales si detecta abuso o riesgo.

5. Contenido del usuario (texto, imágenes, ubicación)

Al usar funciones de chat o reconocimiento, puedes enviar texto, imágenes y (si lo autorizas) ubicación aproximada para sugerir puntos cercanos y rutas.

5.1 Responsabilidad sobre el contenido

Eres responsable del contenido que compartes. No subas información sensible innecesaria (por ejemplo: números completos de identificación, direcciones exactas si no hace falta, datos bancarios, etc.).

5.2 Licencia técnica limitada

Otorgas al Operador una licencia limitada, no exclusiva y revocable para procesar tu contenido únicamente con el fin de operar KataraLM (por ejemplo: generar respuestas, guardar historial, mostrar imágenes en el chat y permitirte acceder a tus propios datos).

5.3 Ubicación

Si habilitas la ubicación, esta se utilizará para mejorar recomendaciones de puntos de acopio y rutas. Puedes desactivar la ubicación en cualquier momento desde tu dispositivo o configuración.

6. IA y limitaciones de responsabilidad

KataraLM utiliza componentes de inteligencia artificial para interpretar entradas y generar respuestas. Aunque buscamos alta calidad, la IA puede equivocarse, estar incompleta o desactualizada.

Seguridad: Para residuos peligrosos (baterías, químicos, aceites, electrónicos, biológicos) sigue instrucciones oficiales, usa contenedores adecuados y consulta gestores autorizados.

6.1 No garantía

La Plataforma se ofrece “tal cual”. No garantizamos exactitud absoluta, disponibilidad ininterrumpida ni resultados específicos.

6.2 Limitación

En la medida permitida por la ley, el Operador no será responsable por daños indirectos, incidentales o consecuentes derivados del uso o imposibilidad de uso de la Plataforma.

7. Seguridad, credenciales y OTP

  • Tu contraseña se almacena de forma segura (hash). No la compartas con nadie.
  • Los códigos OTP expiran y tienen límites de intentos.
  • Si crees que tu cuenta fue comprometida, cambia tu contraseña de inmediato y contáctanos.

El Operador puede implementar medidas adicionales (bloqueo temporal, verificación extra) para proteger a usuarios y la infraestructura.

8. Pagos (si aplica)

Actualmente, KataraLM puede ofrecerse sin costo. Si en el futuro se habilitan planes pagos, se informará claramente: precios, periodicidad, condiciones de cancelación y reembolsos conforme a la normativa aplicable.

9. Servicios de terceros

KataraLM puede integrarse con servicios de terceros para funciones técnicas, como:

  • Envío de correos (p. ej., OTP, notificaciones).
  • Procesamiento de IA para chat y análisis de imágenes.
  • Infraestructura y entrega de contenido (hosting/CDN).
  • Mapas/tiles y geocodificación para visualización de puntos.

Estos terceros pueden tener sus propias políticas. El Operador busca elegir proveedores confiables, pero no controla sus sistemas.

11. Suspensión o terminación
